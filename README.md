## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Backend API endpoint

### Installation

1. Clone the repository:
```bash
git clone https://github.com/saadzaki7/HapiGPTFrontend.git
cd HapiGPTFrontend
```

2. Install dependencies:
```bash
npm install
```

3. Create environment variables:
```bash
cp .env.example .env
```

4. Start development server:
```bash
npm run dev
```
